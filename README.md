# obsidian-vault

OpenClaw + Obsidian 知识记忆库

## 目录结构

```
├── 01-Memory/          # 记忆主目录
│   ├── 000-索引.md     # 索引入口
│   ├── 100-每日/       # 每日记录
│   ├── 200-项目/       # 项目记忆
│   └── 300-知识/       # 知识库
└── 02-Rules/           # 规则目录
```

## 标签体系

- #memory/索引 - 核心索引
- #memory/daily - 每日记录
- #memory/project - 项目相关
- #memory/knowledge - 知识库
- #memory/important - 重要信息

## 技术栈

- OpenClaw
- MCP Adapter
- obsidian-mcp-server
- Obsidian
