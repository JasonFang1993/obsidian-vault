### OpenClaw 避坑指南（翟星人文章）

#### 7个坑
1. 崩溃别让它自己修 - 502等3-5分钟
2. 文件乱放 - 上线建目录结构
3. 复盘流于表面 - 必须回答三问题
4. 不表达观点 - SOUL.md要求主动说
5. 服务器裸奔 - SSH密钥+防火墙
6. 不备份配置 - 改前cp备份
7. 记忆不自动创建 - 需要明确要求

#### 核心观点
- 战友 > 工具人
- 说了 ≠ 做了
- 安全是必选项

---

#### 详细内容补充

##### 坑1：崩溃处理
- 502/503 等3-5分钟大概率自己恢复
- 发 /reset 重置会话
- 不行再 SSH 重启：openclaw gateway stop && restart

##### 坑2：目录结构
```
├── drafts/     # 公众号文章草稿
├── plans/      # 规划文件
├── notes/      # 学习笔记
├── memory/     # 每日短期记忆
├── reviews/    # 每日复盘
├── skills/     # 技能包
└── docs/       # 文档教程
```

##### 坑3：复盘三问题
1. 今天做了什么决策？
2. 哪个决策超出/低于预期？为什么？
3. 如果重来一次，我会怎么做不同？

##### 坑4：要求主动表达
- SOUL.md 明确要求必须表达观点
- 三种情况必须提醒：方向偏了、有更好的方案、有风险

##### 坑5：服务器安全
```bash
# SSH密钥
ssh-keygen -t ed25519
ssh-copy-id root@服务器IP

# 防火墙
systemctl enable --now firewalld
firewall-cmd --permanent --add-port=22/tcp
firewall-cmd --reload

# 禁用密码登录
# 编辑 /etc/ssh/sshd_config
PasswordAuthentication no
PermitRootLogin prohibit-password
```

##### 坑6：配置备份
```bash
cp /root/.openclaw/openclaw.json /root/.openclaw/openclaw.json.bak.$(date +%Y%m%d%H%M)
```

##### 坑7：记忆文件
- 需要在 HEARTBEAT.md 里明确要求才会创建
- 不写就不做

#### 服务器安全检查结果
- 系统：OpenCloudOS 9.4
- 防火墙：云服务器安全组控制，无需本机防火墙
- SSH密钥：已有 ed25519
- 建议：在云控制台配置安全组规则
