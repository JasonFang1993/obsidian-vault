# opencode-caller Skill

## 触发条件

当用户提到以下关键词时自动触发：
- "写代码"、"帮我开发"、"创建 XX 脚本"
- "实现 XX 功能"、"编程"
- ".py"、".js"、".sh" 等代码文件后缀
- "写一个程序"、"帮我写"

## 核心逻辑

1. **自动识别代码需求** - 检测用户是否需要写代码
2. **调用 OpenCode 执行** - 使用 exec command 调用 opencode run
3. **返回结果** - 直接展示代码或执行结果

## 使用方式

```
用户：帮我写一个 Python 脚本来统计文件行数
→ 自动触发 opencode-caller
→ 调用 OpenCode 执行
→ 返回代码结果
```

## 配置方式（可选）

可以在 SOUL.md 或 USER.md 中配置：
- 偏好语言（如主要用 Python/JS）
- 默认工作目录
- 是否自动执行 vs 只返回代码
