## 📦 Khazix-Skills 概览

| Skill | 功能 |
|-------|------|
| **github-to-skills** | 将 GitHub 仓库自动转换为 AI skills |
| **skill-manager** | 管理 skill 生命周期 - 检查更新、列表、删除 |
| **skill-evolution-manager** | 基于用户反馈持续改进 skills |

### 核心功能

1. **github-to-skills**
   - 抓取仓库元数据（README、commit hash）
   - 创建标准化的 skill 目录结构
   - 生成带扩展 frontmatter 的 SKILL.md
   - 用法: `>/github-to-skills <github_url>`

2. **skill-manager**
   - Audit: 扫描本地 skills 文件夹
   - Check: 对比本地与远程 commit
   - Report: 生成状态报告
   - Update: 引导式升级工作流

3. **skill-evolution-manager**
   - Session Review: 分析 skill 表现
   - Experience Extraction: 将反馈转为结构化 JSON
   - Smart Stitching: 将最佳实践持久化到 SKILL.md
