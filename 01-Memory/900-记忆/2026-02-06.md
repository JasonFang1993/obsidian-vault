## HTTP Client Skill v2.2 → v2.3

### Fixed Issues
- Promise 解析错误 → 使用直接 resolve/reject 替代 async/await wrapper
- `parsedUrl.path` → `parsedUrl.pathname` (原生 Node.js 无 `.path` 属性)

### v2.3 新增功能：配置文件
- `.http-client.json` - 保存常用请求
- `.http-client-history.json` - 自动追踪历史请求 (最近50条)
- CLI 命令：`--save`, `--load`, `--list`, `--history`, `--delete`

### Bug Fix
```javascript
// 修复前 (错误)
getSaved(name) {
  return this.saved?.[name] || this.defaults?.[name];
}

// 修复后 (正确)
getSaved(name) {
  return this.config.saved?.[name] || this.config.defaults?.[name];
}
```
