## 2026-02-12 (下午): PlantUML 格式修复 🎯

### 问题
- 初始版本尝试多种方法：raw deflate URL、XML 直接导入
- Draw.io 不支持直接 Mermaid 导入
- 最终采用 PlantUML (SVG) 格式

### 官方示例参考
```
@startuml
skinparam shadowing false
Alice -> Bob: Authentication Request
Bob --> Alice: Authentication Response
@enduml
```

### 最终格式转换
```
输入 (Mermaid):
A[用户发送消息] --> B{AI识别意图}
B -->|问题| C[查询知识库]

输出 (PlantUML):
"用户发送消息" --> (AI识别意图)
(AI识别意图) --> "查询知识库": 问题
```

### 导入方法
1. Draw.io: **Arrange** → **Insert** → **Advanced** → **PlantUML (SVG)**
2. 粘贴 PlantUML 代码
3. 点击 Insert

- AI 可以自测：`call vue-auto-tester.auto_test_vue_project()`
- 自动生成测试报告
