### 技术细节

**MCP 配置**:
```json
{
  "plugins": {
    "entries": {
      "openclaw-mcp-adapter": {
        "enabled": true,
        "config": {
          "servers": [{
            "name": "obsidian",
            "transport": "stdio",
            "command": "npx",
            "args": ["-y", "@mauricio.wolff/mcp-obsidian", "/data/vault"]
          }]
        }
      }
    }
  }
}
```
